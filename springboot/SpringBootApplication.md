# @SpringBootApplication

**Spring Boot 最核心的注解**，整个 Spring Boot 和 Spring Cloud 完整的微服务技术体系，本质上就是围绕着一系列的注解展开的。注解在 Spring 微服务的技术栈中，都扮演着举足轻重的角色和作用。注解的数量非常多，但是 Spring 体系把很多注解都归结到某一个，或者是某几个注解上。使得我们可以更简单的方式去使用注解。
